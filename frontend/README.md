# Sharius Frontend

React frontend for Sharius, an instant cross-device data exchange interface.

## Stack

- React
- Vite
- CSS
- React Router
- QR code generation

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
